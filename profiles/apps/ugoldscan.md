# UGOLDscan

UGOLDscan is a UGOLD-branded blockchain explorer focused on transparent access to Ethereum and BNB Smart Chain data.

## Highlights

- Multi-chain explorer UI for Ethereum and BSC
- Direct RPC-driven architecture
- UGOLD ecosystem integration and supporter metadata
